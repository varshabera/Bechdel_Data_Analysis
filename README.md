# 🎬 Bechdel Test Movie Data Analysis

This project analyzes **female representation in movies** using the **Bechdel Test dataset**.  
The goal is to explore whether movies pass the Bechdel Test and examine relationships between **Bechdel scores, IMDb ratings, and runtime**.

---

## 📊 Project Overview

The **Bechdel Test** measures female representation in films based on three criteria:

1. The movie has **at least two named women**
2. They **talk to each other**
3. Their conversation is about **something other than a man**

This project performs **Exploratory Data Analysis (EDA)** to identify patterns and trends in movie representation.

---

## 🔍 Key Insights

- Around **71% of movies pass the Bechdel Test**
- IMDb ratings show **very little relationship with Bechdel scores**
- Movie runtime does **not significantly affect Bechdel results**
- Correlation between variables is **very weak**
